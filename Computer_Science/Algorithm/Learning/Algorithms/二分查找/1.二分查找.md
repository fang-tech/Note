# 二分查找的实质

- 通过最开始简单的题目, 我们只是知道可以通过二分查找在一个有序的数组中快速地锁定我们需要的数字, 但是却没有认真地剖析过整个思路过程

## 单调性

> 单调性可以说是二分查找的最核心的地方了, 就是因为有单调性, 我们才能有check函数, 用于确认mid位置是在正确答案的右边还是左边

- 什么是单调性
  - 这里我们能给出一个非形式化的定义, 对于当前的情景, 满足x(我们需要求得数据) 越大(小), check函数越返回false, 反之则越返回true
  - 这样我们就能利用单调性, 每次都能确认二分之一的位置

## check函数

> 这个函数算是二分的复杂题目里面最难写出来的部分了, 我们现在给出了一个答案x, 我们要能通过check函数验证这个x可行的还是不可行的

- 这里我们需要明确的一点是, **我们不是在推演, 只是在验证!**, 但是这个点也是很难形式化说明的点, 详细的例子可以看2861

- 可以说写二分的题目, 基本就是在写check, 只不过一些check函数比较简单, 我们甚至没有必要单独写一个函数出来罢了

## 模板代码

```cpp
    bool check(){}

    int function() {

        // true
        int left = lower_bound - 1;
        // false
        int right = upper_bound + 1;
        while (left < right - 1) {
            int mid = left + (right - left) / 2;
            (check() ? left : right) = mid;
        }
        return left;
    }
```
