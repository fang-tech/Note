# 做题模板

> 树与递归

由树的结构性, 其实很容易的出来, 递归是树的一个很漂亮的遍历方式, 因为 **原问题和子问题具有相同的结构**

> 做题的时候, 抽象出来一个问题, 将问题分解成原问题和子问题, 然后思考如何由子问题组合成原问题, 再处理边界条件和返回的时机
